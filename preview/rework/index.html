Enhancing Cultural Heritage Knowledge

Sandro Botticelli in the ArCo Knowledge Graph

KE4H project — knowledge graph exploration, gap analysis and proposed RDF enrichment

Abstract

This project explores how Sandro Botticelli is represented in the ArCo Knowledge Graph. Using SPARQL queries against the ArCo SPARQL endpoint, we examine the RDF description of a selected artwork and compare structured triples with information that appears only in the labels of related photographic resources.

The analysis identifies a knowledge gap concerning philosophical and interpretative context (notably Neoplatonism). Large Language Models are used to support gap identification and to compare alternative enrichment strategies. The project proposes new RDF triples using existing ArCo properties and discusses validation, challenges and the respective strengths of KG-based and LLM-based approaches.

Author

Kseniia Schnaider — individual project (KE4H).

Main ArCo resource

https://w3id.org/arco/resource/HistoricOrArtisticProperty/1201008107

Madonna con Bambino, San Giovannino e angeli by Sandro Botticelli (HistoricOrArtisticProperty).

Research questions

  • Which Botticelli-related entities are present in ArCo and how is the main artwork described in RDF?
  • Is interpretative / philosophical context represented through structured properties or only indirectly in photographic labels?
  • Can LLMs help identify enrichment candidates, and how do their suggestions compare with SPARQL evidence?
  • How can the knowledge graph be enriched using existing ArCo vocabulary?

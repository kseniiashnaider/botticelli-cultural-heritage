Validation

The proposed enrichment was not inserted into the production ArCo endpoint. Validation is therefore conceptual: we define SPARQL queries that would retrieve the new knowledge if the triples from RDF Enrichment were added to the graph.

Before enrichment — Query 6 (baseline)

Query 6 returns no direct links to philosophical entities. This is the baseline against which enrichment is measured.

After enrichment — Future query: subjects of the painting

After adding the proposed triples, the following query would return Neoplatonism as a subject:

PREFIX a-cd: <https://w3id.org/arco/ontology/context-description/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT ?subject ?label
WHERE {
  <https://w3id.org/arco/resource/HistoricOrArtisticProperty/1201008107>
      a-cd:hasSubject ?subject .
  OPTIONAL { ?subject rdfs:label ?label . }
}

Placeholder. Expected after enrichment: ex:Neoplatonism with labels "Neoplatonism"@en / "Neoplatonismo"@it.

After enrichment — Future query: involved agents

PREFIX core: <https://w3id.org/arco/ontology/core/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT ?agent ?label
WHERE {
  <https://w3id.org/arco/resource/HistoricOrArtisticProperty/1201008107>
      core:involvesAgent ?agent .
  OPTIONAL { ?agent rdfs:label ?label . }
}

Placeholder. Expected: Marsilio Ficino as core:Agent.

Validation summary

The future queries show that enrichment would make philosophical context retrievable through standard ArCo properties, without relying on REGEX over photographic labels. Compared with the previous project draft (a catalogue authorship query unrelated to Neoplatonism), these queries directly test the proposed triples.

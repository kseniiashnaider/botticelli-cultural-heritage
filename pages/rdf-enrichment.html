Sandro Botticelli in the ArCo Knowledge Graph

RDF Triples and Proposed Enrichment

This section presents the RDF triples proposed to enrich the representation of the selected Botticelli painting in the ArCo knowledge graph. The enrichment addresses gaps identified through SPARQL and LLM analysis.

Main resource

https://w3id.org/arco/resource/HistoricOrArtisticProperty/1201008107

Enrichment strategy

  • Gap 1: philosophical / interpretative context appears in photographic labels but not as structured subjects
  • Gap 2: no direct triples link the painting to Neoplatonism

The enrichment reuses existing ArCo properties from the Context Description and Core modules.

Vocabulary reused

  • a-cd:hasSubject — links the cultural property to a subject resource
  • a-cd:Subject — types the subject entity
  • core:involvesAgent — optional link to Marsilio Ficino as intellectual agent
  • rdfs:label — bilingual labels for new resources

SPARQL CONSTRUCT query

PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX a-cd: <https://w3id.org/arco/ontology/context-description/>
PREFIX core: <https://w3id.org/arco/ontology/core/>
PREFIX ex: <https://example.org/botticelli-enrichment/>

CONSTRUCT {
  <https://w3id.org/arco/resource/HistoricOrArtisticProperty/1201008107>
      a-cd:hasSubject ex:Neoplatonism ;
      core:involvesAgent ex:MarsilioFicino .

  ex:Neoplatonism
      rdf:type a-cd:Subject ;
      rdfs:label "Neoplatonism"@en ;
      rdfs:label "Neoplatonismo"@it .

  ex:MarsilioFicino
      rdf:type core:Agent ;
      rdfs:label "Marsilio Ficino"@en ;
      rdfs:label "Marsilio Ficino"@it .
}
WHERE { }

Interpretation

Instead of introducing a custom predicate such as influencedBy, the enrichment models Neoplatonism as an a-cd:Subject linked through a-cd:hasSubject, following the modelling pattern illustrated in the KE4H course materials (e.g. subject representation for cultural properties). Marsilio Ficino is included as an optional core:Agent connected to the intellectual context of Botticelli's Florence.

Modelling decision

An initial version of this project used ex:influencedBy on the example.org namespace. After reviewing the ArCo ontology network, that approach was replaced: a-cd:hasSubject and core:involvesAgent are already defined for cultural properties and agents. Custom properties were therefore not introduced. Only new URIs under ex: identify specific subject and agent resources missing from the original RDF description.

Why these triples improve the knowledge graph

  • Philosophical context becomes directly queryable via a-cd:hasSubject
  • The enrichment is aligned with ArCo's Context Description module
  • Future SPARQL queries can retrieve subjects without parsing photographic labels

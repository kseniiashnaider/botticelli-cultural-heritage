Challenges and Solutions

16. Project Limitations

The enrichment presented in this project is intentionally illustrative. Some proposed RDF properties belong to an example namespace because equivalent concepts are not currently available in the ArCo ontology. Only one artwork was analysed, and only a limited set of LLMs was evaluated. Future work could align the enrichment with CIDOC CRM, Iconclass and Getty vocabularies and test the workflow on larger cultural heritage collections.

4.9 Limitations (RDF Enrichment)

The illustrative properties defined within the ex: namespace are introduced solely for demonstration purposes because equivalent properties are not currently available in the explored ArCo model. Before integration into a production knowledge graph, all proposed RDF statements should be validated by domain experts and, where appropriate, aligned with established ontologies such as CIDOC CRM or Iconclass.
